Introduction

This web-app makes organizing home tasks possible. This web-page is divided into different apps:

  • Tasks
  • Groceries
  • Finances
  • Meal-planning
Each of those apps organizes different areas. In the following cards, each of those areas and the possible functions of those tasks are explained further.

Tasks/Calendar

The task app offers multiple functions:

  • Scheudle Tasks
  • Recurring Tasks
  • Planned Events

Groceries

The groceries app helps organize grocery shopping. Furthermore bought items are tracked and sorted after categories so that the data can be analysed and visualized later on.
